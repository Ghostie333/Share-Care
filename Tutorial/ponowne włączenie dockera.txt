1. Wejść do folderu gdzie jest docker-compose.yml
2. Otworzyć tam powershella
3. wpisać: docker-compose up -d


Development:
docker compose -f docker-compose.dev.yml down -v (-v kasuje dane)
docker compose -f docker-compose.dev.yml up -d --build


Prod:
docker compose -f docker-compose.prod.yml down -v
docker compose -f docker-compose.prod.yml up -d --build
